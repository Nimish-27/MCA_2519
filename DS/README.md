# Data Structures

This directory contains implementations and examples of various data structures.

## Topics Covered

- **Arrays**: Basic array operations and algorithms
- **LinkedLists**: Singly, doubly, and circular linked lists
- **Stacks**: LIFO data structure implementations
- **Queues**: FIFO data structure implementations including circular and priority queues
- **Trees**: Binary trees, BST, AVL, and other tree structures
- **Graphs**: Graph representations and algorithms
- **Searching**: Linear search, binary search, and other search algorithms
- **Sorting**: Bubble sort, quick sort, merge sort, and other sorting algorithms
- **Hashing**: Hash tables and collision handling techniques
